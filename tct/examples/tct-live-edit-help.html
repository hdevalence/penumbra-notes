Welcome to the Tiered Commitment Tree!

Within the demo, you can edit the tree using keyboard commands:

  • c inserts a random commitment without remembering it
  • ⇧C inserts a random commitment and remembers it
  • b ends the current block
  • ⇧B inserts a random block root
  • e ends the current epoch
  • ⇧E inserts a random epoch root
  • f forgets a random commitment
  • r evaluates the root of the tree
  • n resets the tree to new
  • ? shows this help text again

Prefix a command key with a number to repeat it, vim-style. For example, 3f will forget three commitments randomly.

Scroll and drag the page to zoom and pan across the tree.

If you mouse over an epoch, block, commitment, or hash, the tooltip will show more info.
