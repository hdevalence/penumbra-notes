How to Contribute

Creating a Wallet

The first step to contributing is to setup a wallet on the command line. To do this, follow the Penumbra Guide in order to setup pcli and create a wallet.

Receiving Faucet Funds

After setting up a wallet, you can join our Public Discord in order to get testnet funds.
To do this, you need to first get your address from pcli: 

cargo run --quiet --release --bin pcli -- v address
Then, you can go to the #testnet-faucet channel in the Discord, and use that address to receive funds.

Contributing

Foo bar baz. You can then join the contribution queue for the current phase with the following command: 

cargo run --quiet --release --bin pcli -- ceremony contribute --phase 1 --bid {{ min_bid }} --coordinator-address {{ address }}
Once in the queue, you just have to wait until the command completes.
You can improve your position in the queue by bidding more funds. Note that each time you bid with the command above, this adds additional funds to all of the previous bids you've already submitted. If you get disconnected for some reason, you can simply bid 0penumbra and recover your position in the queue, since all of your previous bids have been recorded. Note as well that you need to bid at least {{ min_bid }} to contribute.
